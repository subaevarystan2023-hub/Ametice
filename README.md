# 🏎️ Speed Rush

**Speed Rush** — это 3D гоночная игра, разработанная в Unity 6 на языке C# в рамках дипломного проекта.

---

## 🎮 Описание игры

Вы садитесь за руль гоночного автомобиля и проходите трассу с препятствиями на время. После финиша игра сохраняет ваш лучший результат. Сможете ли вы побить свой рекорд?

---

## 🕹️ Управление

| Действие | Клавиши |
| Газ / Тормоз | W / S или ↑ / ↓ |
| Поворот | A / D или ← / → |
| Переключение камеры | указано в игре |
| Пауза | Escape |

---

## 🗂️ Сцены

| Сцена | Описание |
| **MainMenu** | Главное меню с кнопками Play / Exit и отображением лучшего результата |
| **Race** | Основная гоночная сцена — трасса, машина, отсчёт 3-2-1, таймер |
| **Result** | Экран результатов — Your Score, Best Score, New Best |

---

## ⚙️ Механики

- 🚗 Физика автомобиля (Rigidbody, Wheel Collider)
- ⏱️ Секундомер гонки — засекает время от старта до финиша
- 💾 Сохранение рекорда через **PlayerPrefs**
- 🎥 Система камер с несколькими режимами обзора (Far, Normal, FP)
- 🏁 Система чекпоинтов и кругов
- 🔢 Обратный отсчёт 3-2-1 перед стартом
- 🛞 Анимация колёс и следы от шин
- 📊 Экран результатов с отображением лучшего времени

---

## 🛠️ Технологии

- **Unity 6** (6000.3.5f2)
- **C#**
- **TextMeshPro** — UI текст
- **Rigidbody / Wheel Collider** — физика машины
- **PlayerPrefs** — хранение данных
- **SceneManager** — переходы между сценами
- **Animator** — анимации
- **Racing Starter Kit (SpinMotion)** — базовая система гонки

---

## 📁 Структура проекта

```
Assets/

```

## 🚀 Как запустить

1. Клонируйте репозиторий:
bash
git clone https://github.com/subaevarystan2023-hub/Ametice.git
```
2. Откройте проект в **Unity Hub** → версия Unity 6
3. Откройте сцену `Assets/Scenes/MainMenu`
4. Нажмите **Play**
